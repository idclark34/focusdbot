FocusdBot Explained

A complete beginner's guide to understanding how FocusdBot works, what it does, and how it's built

What is FocusdBot? 🤔

🤖

Your Focus Companion

FocusdBot is a cute robot that lives in your Mac's menu bar and helps you stay focused while working. Think of it as a friendly productivity coach that watches over your computer habits.

⏱️

Pomodoro Timer

It uses the famous Pomodoro Technique - work for 25 minutes, then take a 5-minute break. The robot shows a countdown timer and celebrates when you complete a session!

👀

Smart Monitoring

The robot can detect when you switch to distracting apps (like social media) and gently reminds you to get back to work by changing colors and expressions.

How Does It Work? ⚙️

1

You Start Working

Click the robot to start a 25-minute focus session. The robot turns blue and shows a countdown timer.

2

Robot Watches

FocusdBot monitors which apps you're using. It knows the difference between work apps and distracting ones.

3

Gentle Reminders

If you switch to a distracting app, the robot turns red and looks angry - a gentle nudge to get back to work!

4

Celebration Time

Complete your 25 minutes and the robot turns green, shows confetti, and asks you to reflect on what you accomplished!

Meet Your Robot's Moods 😊

😴

Idle

Waiting for you to start a focus session. The robot breathes gently and wiggles occasionally.

😊

Focused

You're working! The robot is happy, shows a blue color, and displays your remaining time.

😠

Distracted

Uh oh! You're on a distracting app. The robot turns red and pulses to get your attention.

🎉

Success!

You did it! The robot turns green, shows confetti, and celebrates your completed focus session.

Cool Features 🚀

📱

iPhone Integration

Connect your iPhone! If you pick up your phone during a focus session, the robot will know and remind you to put it down.

🌐

Website Blocking

Block distracting websites in Safari! Add sites like Reddit or Twitter to your blocked list during focus time.

📊

Beautiful Analytics

See how much time you spent focused vs distracted, which apps steal your attention most, and track your progress over time.

🎨

Customizable

Choose different timer display styles (neon, gradient, dot matrix), adjust durations, and pick which apps are allowed during focus time.

📝

Session Reflections

After each session, reflect on what you accomplished. The app builds a history of your focus sessions and insights.

🔒

Privacy First

Everything stays on your Mac! No data is sent to the internet. Your focus habits are private and secure.

How is it Built? 🔧

Technical Architecture

User Interface (SwiftUI)
The cute robot, timer displays, menu bar, and settings screens
App Monitoring (macOS APIs)
Watches which apps you use and detects when you switch between them
Data Storage (SQLite Database)
Remembers your sessions, app usage, and statistics over time
System Integration (Accessibility & AppleScript)
Connects to Safari for website blocking and gets window information

🖥️ User Interface

  • SwiftUI Modern
  • AppKit macOS
  • Menu Bar Integration Native
  • Floating Robot Window Custom

🧠 Intelligence

  • NSWorkspace API App Tracking
  • Accessibility APIs Window Info
  • AppleScript Safari
  • Real-time Monitoring Live

💾 Data & Storage

  • SQLite Database Local
  • GRDB Framework Swift
  • UserDefaults Settings
  • Migration System Updates

📱 Connectivity

  • MultipeerConnectivity iPhone
  • Peer-to-Peer Network Local
  • Real-time Sync Instant
  • No Internet Required Offline

How to Get Started 🚀

📥

1. Download & Install

Run the install script to build and install FocusdBot to your Applications folder.

./install.sh
🔑

2. Grant Permissions

Allow Accessibility access so FocusdBot can monitor your apps, and AppleScript access for Safari integration.

🎯

3. Start Focusing!

Click the robot to start your first 25-minute focus session. Configure allowed apps and websites in the menu bar.

Quick Test Mode

New to FocusdBot? Try the 30-second test mode first!

Click menu bar → Quick Test → 30s → Start Timer
